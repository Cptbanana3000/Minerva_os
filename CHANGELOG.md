# Changelog

All notable MinervaOS milestone changes should be recorded here.

This project is still pre-release, so entries are grouped by development
snapshot rather than stable semantic versions.

## Unreleased

- Refreshed public README and documentation.
- Added architecture, contributing, roadmap, release checklist, and license
  decision notes.
- Added GitHub issue and pull request templates.
- Added MIT license.
- Started Phase 10 with a VBE / linear framebuffer implementation plan.
- Added package metadata commands: `pkg list/info/install/status/remove`.
- Added app registry commands: `app list/info/run/add/remove/check`.
- Added in-OS SDK reference through `sdk` and `SDK.TXT`.
- Expanded the bootloader kernel load window to 192 KiB.
