# Support

MinervaOS is currently a development project, not a supported product.

## Getting Help

- Read [README.md](README.md) for build and run instructions.
- Read [PHASES.md](PHASES.md) for the roadmap.
- Read [PROGRESS.md](PROGRESS.md) for detailed milestone history.
- Use GitHub issues for reproducible bugs or focused feature requests once the
  repository is public.

## Good Bug Reports Include

- Host OS and QEMU version.
- Exact command typed in MinervaOS.
- What appeared on screen.
- Serial output if available.
- Whether `make -B` and `git diff --check` pass.
