## Current Focus

MinervaOS is finishing Phase 9 polish and open-source preparation:

- package metadata
- app registry and SDK seed
- documentation
- release checklist
- public contribution workflow

## Next Engineering Direction

After Phase 9, the likely next major engineering arc is Phase 10: moving beyond
320x200 VGA toward VESA/linear-framebuffer graphics, higher resolutions,
true-color drawing, scalable UI, and crisp icons.
