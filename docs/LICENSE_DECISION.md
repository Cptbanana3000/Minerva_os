## Common Options

- MIT: short, permissive, easy for others to use in many contexts.
- BSD-2-Clause or BSD-3-Clause: permissive, similar spirit to MIT.
- GPL-2.0-or-later or GPL-3.0-or-later: copyleft; derivatives must preserve
  the same license terms.
- MPL-2.0: file-level copyleft with more permissive linking behavior.

## Release Notes

- The full license text is in `LICENSE`.
- New source files may optionally include an SPDX header:
  `SPDX-License-Identifier: MIT`.
- Third-party code or assets should be reviewed before inclusion so their
  licenses are compatible with MIT distribution.
